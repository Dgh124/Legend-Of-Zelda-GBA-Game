Defeat Ganon and Save Hyrule

How the game works:
- The purpose of this game is to defeat Ganon and save Hyrule
- If Link touches Ganon you automatically lose
- Ganon shoots fireballs at link over time. Each time one hits Link, he loses 1 heart.
- You lose the game if Link runs out of hearts, or the timer hits 0.
- You win the game by shooting your sword at Ganon until his health reaches 0.
- Once you beat the game, you can replay in either normal mode (pressing enter) or hard mode (pressing A) from the start screen. 
- In hard mode, link has less health, Ganon moves faster, fireballs move faster, and the timer starts at 30 seconds.
- There is a 1 second cooldown on the master sword.
- The start screen will display your personal best time. Higher times indicate beating the game quicker.

How to play:
- Press enter to start the game from the welcome screen
- Use the arrows to move link around
- Press A (x on Mac) to shoot the master sword
- Press backspace to restart the game at any time

To run this game:
- download the zip file
- navigate into folder hw08_macos in terminal in docker
- run command ->  make mgba
